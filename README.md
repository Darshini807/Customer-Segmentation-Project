# Customer Segmentation Project

## Objective
Segment customers based on purchasing behavior and demographics.

## Features
- Customer data analysis
- Customer segmentation into Low, Medium, and High Value groups
- Spending pattern analysis
- Dashboard visualization using charts

## Tools Used
- Microsoft Excel

## Segments
### Low Value Customers
- Low spending
- Few purchases

### Medium Value Customers
- Moderate spending
- Regular purchases

### High Value Customers
- High spending
- Frequent purchases

- Customer_Segmentation.xlsx

- Customer_Segmentation_Report.docx

## Outcome
Successfully analyzed customer behavior and created customer segments with visual insights.
